# 🧺 Laundry Management System (Real-time & Payment Gateway)

Sistem pengurusan dobi moden yang menyokong pembayaran automatik melalui Midtrans, notifikasi real-time menggunakan Laravel Reverb, dan aplikasi mobile berasaskan Expo.

## 🚀 Prasyarat (Prerequisites)
Pastikan laptop anda sudah terpasang:
- **PHP >= 8.2** & **Composer**
- **Node.js** (LTS version) & **npm/yarn**
- **MySQL** (XAMPP/Laragon)
- **Ngrok** (Untuk webhook Midtrans)
- **Expo Go** (Pada telefon pintar anda)
